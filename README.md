# rock-paper-scissors

TOP Rock paper scissors console only game
This is a game run on a browser console. It takes a user's prompt of three choices (rock, paper scissors) and returns a result played against a computer.

The computer's choice is decided by the Math.Rand() function, and is assigned a random value depending on the result.

The user and computer's choices are compared to one another and allows for the game to progress via comparison conditions. It keeps track of the round, the score and the rounds played in case of a tie. The winner is declared at the end.
